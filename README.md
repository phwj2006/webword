# 打字记单词

中小学英语单词练习应用，支持打字记忆、默写、艾宾浩斯复习、错词库。

## 教材版本

### 小学（3-6年级）
- 人教精通版（天津）- 1039 词
- 冀教版（河北）- 735 词

### 初中（7-9年级）
- 人教版初中 Go for it!（天津）- 421 词
- 冀教版初中（河北）- 137 词

共 **2332** 个词条。

## 功能
- 每日学习新词，自动安排艾宾浩斯复习
- 打字练习：实时逐字符高亮反馈
- 默写练习：凭音标和释义拼写，支持自动发音
- 错词库：自动收集练习中的错词
- 学习统计：打卡日历、连续天数、进度追踪
- 管理后台：用户管理、学习数据查看

## 技术栈
- 前端：纯 HTML/CSS/JS（无框架依赖）
- 后端：Python 标准库（http.server + sqlite3），零依赖
- 部署：Docker

## 本地开发
```bash
python server.py
# 访问 http://localhost:8000
```

## Docker 部署
```bash
docker compose up -d
# 访问 http://localhost:8000
```
